# AI Context Documentation

This folder contains documentation and prompts designed for AI assistants working with the Clarion Extension codebase.

## Contents

### Startup Prompts
- **STARTUP_AI_PROMPT.md** - Main startup prompt with project context and guidelines
- **TIER1_MINIMAL_PROMPT.md** - Minimal tier-1 prompt for quick context loading
- **SESSION_RESUME_2025-12-16.md** - Session resume document with ongoing work context

### Feature Context
- **ClarionExtensionFeatures.md** - Comprehensive feature list and implementation details for AI reference

## Purpose

These documents provide:
- **Project overview** and architecture for AI assistants
- **Coding guidelines** and patterns used in the extension
- **Feature descriptions** with implementation details
- **Context continuity** across AI sessions

## Usage

When starting a new AI session working on this extension:
1. Load the appropriate startup prompt (STARTUP_AI_PROMPT.md or TIER1_MINIMAL_PROMPT.md)
2. Reference ClarionExtensionFeatures.md for feature-specific context
3. Use session resume documents to continue previous work
